
  # Eurasian Railway History Viewer

  This webpage is a tool for the internal use during the construction of Eurasian Railway History database. It plots events connected with segments and stations in time.

  Live webpage can be accessed [here](https://jjkiljanski.github.io/eurasian-railway-history-viewer/).

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server locally.
